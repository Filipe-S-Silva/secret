# 💌 Convite Secreto

Um site interativo e minimalista desenvolvido com HTML, CSS e JavaScript. O projeto simula o envio de uma carta digital que, após a interação, revela um convite especial.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura das páginas.
- **CSS3**: Estilização e animações de abertura da carta.
- **JavaScript**: Lógica de interação, verificação de nome e redirecionamento.

## 🎨 Funcionalidades

- **Página 1 (Envelope)**: Uma interface visual de uma carta fechada.
- **Página 2 (Convite)**: Uma transição suave para a revelação do convite especial.
